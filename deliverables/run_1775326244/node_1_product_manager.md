Calculator Application Specification

Functional Requirements

Display Area: A dedicated top section to show the current numeric input, the active mathematical operation, and the final calculated result.
Numeric Keypad: Ten distinct buttons representing the digits 0 through 9.
Operators: Four dedicated buttons for basic arithmetic operations, specifically addition, subtraction, multiplication, and division.
Action Buttons: An equals button to execute the calculation, a clear button to completely reset the current input and memory, and a decimal point button to allow for fractional numbers.

User Interface and Design Language

Overall Aesthetic: Ultra-modern, premium, and futuristic layout.
Glassmorphism: The main calculator container and individual buttons must utilize a frosted glass effect. This includes semi-transparent backgrounds with a subtle background blur and light borders to blend seamlessly with any background environment.
Neon Accents: Vibrant, glowing elements should be used to establish visual hierarchy. Key items like the display border, active operators, and the equals button should feature striking neon colors with a soft drop-shadow glow.
Geometry: A soft, approachable structural design utilizing heavily rounded corners on both the main calculator chassis and all interactive buttons.
Interactive Elements: Smooth hover effects driven by fluid transition animations. When a user hovers over or taps a button, it should exhibit a gentle upward scale and an intensified neon glow, providing immediate and satisfying visual feedback.
