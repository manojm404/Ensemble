# Changelog – Ensemble AI OS

All notable changes to the Ensemble platform will be documented in this file.

## [v1.5.0] – Agentic OS Bridge (Latest)
### Added
- **Searchable Specialist Directory**: 20+ expert personas (CEO, CTO, Legal, PM, etc.).
- **Advanced Command Input**: Auto-expanding chat terminal with Tool Ribbon (Attachments, Web Search, Docs, @Mentions).
- **Dynamic Persona Injection**: Deterministic specialist routing based on assistant selection.
- **WebSocket Streaming**: Real-time broadcast of Gemini reasoning events.
- **UI Logic**: Click-outside detection for agent popovers and `@mention` shortcut triggers.

## [v1.0.0] – Ensemble Core Engine
### Added
- **SOP Engine**: Finite State Machine (FSM) executor for agent workflows.
- **Managed Agents**: Autonomous execution loop for complex tasks.
- **Governance Service**: SQLite-backed approval queue and cost-tracking.
- **Audit System**: Unified real-time event logging.
- **CAS Space**: Content-Addressable Storage for agent artifacts and memory.
- **LLM Provider**: Unified interface for Gemini, Ollama, and local models.
